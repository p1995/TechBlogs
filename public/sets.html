Sets

What are Sets?

Sets are similar to map except they store unique values, not key-value pairs. Example {1,2,3}

Most common Sets methods:

  1. add(value): Adds a value.
    mySet.add(1);
  2. delete(value): Removes a value.
    mySet.delete(1);
  3. has(value): Checks if a value exists.
    mySet.has(1);
  4. clear(): Removes all values.
    mySet.clear();
  5. size: Returns the number of values.
    mySet.size;
  6. keys() / values(): Returns an iterator of values.
    for (const val of mySet.values()) console.log(val);
  7. entries(): Returns [value, value] pairs.
    mySet.entries();
  8. forEach(callback): Executes a callback for each value.
    mySet.forEach(val => console.log(val));
