Specificity

What is Specificity?

CSS specificity determines which rule is applied when multiple rules target the same element. It is calculated based on the type of selectors used, with a hierarchy from least to most specific:

Type selectors (element selectors):

These have the lowest specificity. Example: div { color: red; }

Class selectors:

More specific than type selectors. Example: .box { color: blue; }

Attribute Selector:

More specific than Class. Example div[draggable] { color:orange; }

ID selectors:

Even more specific than Attribute selectors. Example: #header { color: green; }

Inline styles:

The most specific, applied directly in the HTML element. Example: div style="color: yellow;"

Specificity Order (from least to most specific):

Type selectors
Class selectors
ID selectors
Inline styles
